# Vector

> Code with direction.

Vector is a modern AI-assisted code editor inspired by Cursor.  
It focuses on speed, clarity, and intelligent developer workflows.

Built with a minimal interface and powerful AI integration, Vector aims to reduce friction between thinking and coding.

---

## ✨ Features

- ⚡ Fast and lightweight editor
- 🧠 AI-assisted code generation and refactoring
- 💬 Inline explanations and suggestions
- 🪄 Prompt-to-code workflows
- 🌙 Dark mode support
- 🎯 Clean, distraction-free UI
- 🔌 Extensible architecture (planned)

---

## 🛠 Tech Stack

- **Frontend:** React / Next.js
- **Editor Engine:** Monaco Editor
- **Backend:** Node.js / Express
- **AI Integration:** Ai.sdk, 
- **Database:** Convex
- **Styling:** Tailwind CSS
